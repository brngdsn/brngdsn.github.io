# Braun Goodson - Software Engineer Portfolio

Welcome to my personal GitHub page! This repository hosts the code for my professional portfolio, showcasing my work as a software engineer specializing in web and mobile development.

## Live Website
Check out my portfolio live at: [brngdsn.github.io](https://brngdsn.github.io)

## Overview

This webpage presents my career achievements, skills, and interests in software development, including my experience in:

- Full-Stack Development (Web & Mobile)
- Cloud Solutions & Serverless Architectures
- API Development & Integration
- UI/UX Design
- AI, LLMs, and Blockchain Technologies

## Key Sections

- **Home**: A brief introduction to who I am and what I do.
- **About**: Learn more about my background, skills, and areas of expertise.
- **Resume**: A detailed look at my professional experience, education, and career milestones.
  
## Technologies Used

The portfolio is built using the following technologies:

- HTML5 & CSS3
- [Bootstrap 5](https://getbootstrap.com/)
- JavaScript & jQuery
- Vendor Libraries:
  - [Purecounter](https://github.com/srexi/purecounterjs) (for counters)
  - [Bootstrap Icons](https://icons.getbootstrap.com/) (for icons)
  - [Swiper.js](https://swiperjs.com/) (for slideshows)
  - [GLightbox](https://biati-digital.github.io/glightbox/) (for image galleries)
- Google Fonts (Open Sans, Raleway, Poppins)
  
## Features

- **Responsive Design**: The site is optimized for viewing on both desktop and mobile devices.
- **Interactive Skills Section**: Dynamic progress bars showcasing my proficiency in key areas.
- **Favourite Quotes**: A collection of quotes from influential figures in tech and beyond.
- **Professional Summary**: Insight into my work across industries like healthcare, finance, and insurance.


## Getting Started

To view or modify the portfolio locally:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/your-repository.git
   ```
2. Navigate to the project directory:
   ```
   cd your-repository
   ```
3. Open `index.html` in your browser to view the site locally.
   ```
   open index.html
   ```

Alternatively, you can serve the project using a local web server like `http-server` for better performance:

1. Install `http-server` globally if you haven't already:
   ```
   npm install -g http-server
   ```
2. Serve the project:
   ```
   http-server .
   ```
3. Open your browser and go to the displayed local URL (e.g., `http://localhost:8080`).
