Idiom IAM Query

Use Case

As a data engineer, I should be able to choose an s3 action and resource, so that I can see which roles and associated policies I need for my ETL process.



Architecture



Challenges

AWS IAM Roles, Policies, Groups, and Users can have complex relationships, specifically when each of those entities can have their own set of managed or attached overriding rules. To build these relationships, the RESTful API layer had to consume over a dozen of the IAM endpoints, and join the information together. Since the information was inherently document oriented, there was a linear path from endpoint to endpoint to build the model, and so using ESNext features like async and await keywords was essential to handle the promise based strategy used in the application.



Skills and Competencies

  • Firm understanding of IAM API and AWS SDK
  • Firm understanding of RESTful API design and consumption
  • Use functional programming paradigm and ES6 for promise based asynchronous http requests/responses in NodeJS back-end, and VueJS front-end
  • Use in-memory SQL DB
  • Lead the path for development
  • Save team time (which equals money)
API Explorer

Visualize Open Data APIs with Ease

Literally thousands of Open Data APIs exist to date, and to explore these APIs takes a lot of unrelated knowledge like understanding RESTful API Programming. Even just for simple curiousity, each API can vary from protocol, DTO, and attribute types. This API Explorer can digest any JSON or CSV API. There is a streaming RDJSON proxy in the case of APIs with no support for CORS. The tool can fetch and stream JSON and CSV information from any endpoint, and then attempt to analyze the response and recognize simple object attributes. For instance, exploring the paleobiology data api, objects returned can have a `img` attribute, and a pair of `lng` and `lat` attributes representing a map. The tool can recognize these attributes, and give a rich experience in response, like showing a google map, or an image.

Daily NASA

NASA Daily

The Open Data APIs at NASA provider a daily image of space coupled with some meta data description. Using a secret API key, this browser client makes asynchronous RESTful calls to the NASA server. It's written in VueJS as a stateless SPA-oriented front-end, with vuex and vue-resource, and bulma.
